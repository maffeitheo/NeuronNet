Detailed Description

This is the main class. It manages user inputs, defines the simulation parameters and constructs the Network net. It then runs the simulation and prints the results to the output streams. These streams can be files with names based on the string output with a suffix.

Simulation parameters:

  • endtime : total number of time-steps,
  • size : total number of neurons,
  • degree : average connectivity of a neuron,
  • thalam : st. dev. of thalamic input (for excitatory neurons),
  • streng : average intensity of connections,
  • inhib : fraction of inhibitory neurons in the network,

The map ntypes describes the neuron population: its keys are the neuron types from Neuron::NeuronTypes and values are the corresponding counts.
